## 2. 미처리 개발 — 코드로 해결 가능 (자율 진행 대상)

우선순위 순. 완료 시 [x] 체크.

- [ ] **D-1. 상세 화면 지도 방어적 렌더** — 유효한 Maps 키가 없을 때 `GoogleMap`을 렌더하지 않고
  주소/좌표 카드만 표시(키 있을 때만 지도). 상세 화면 크래시/빈 지도 동시 해결.

- [ ] **M-3. 미사용 문자열 `detail_ai_suggest` 제거** (AI 작업에서 잔존).
