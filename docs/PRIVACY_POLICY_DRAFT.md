# QuietKeeper 개인정보처리방침 (초안)

> ⚠️ **초안** — 출시 전 법률 검토 필요. Google Play 등록 시 이 문서를 공개 URL로 호스팅하고
> 스토어 콘솔의 "개인정보처리방침 URL"에 등록해야 한다. 마이크 녹음·위치 수집 앱은 필수.

최종 수정: 2026-06-21

## 1. 수집하는 정보

| 항목 | 수집 이유 | 저장 위치 |
|---|---|---|
| **오디오 녹음(원음 WAV)** | 기준 초과 소음의 객관적 증빙 | 기기 로컬(앱 전용 저장소). 프로/클라우드 동의 시 클라우드 |
| **소음 측정값(dB(A), Leq, Lmax)** | 측정 기록·통계 | 기기 로컬 DB. 동의 시 클라우드 |
| **측정 위치(GPS 좌표·주소)** | "측정이 이뤄진 장소" 증빙 | 기기 로컬. 동의 시 클라우드 |
| **기기 식별값(ANDROID_ID)** | 데이터 무결성 해시(SHA-256) 생성 | 해시 입력으로만 사용, 원본 미저장 |
| **이동 감지(가속도/자이로)** | 부정 측정 방지 플래그 | 기기 로컬(플래그만) |
| **결제 정보** | 프로 구독 처리 | Google Play가 처리(앱은 영수증 토큰만) |

광고 식별자(AdMob): 무료 플랜에서 광고 표시를 위해 Google AdMob SDK가 광고 ID를 사용할 수 있음.

## 2. 이용 목적
- 층간소음 측정·기록 및 사용자 본인의 증빙 자료 생성
- (동의 시) 클라우드 백업 및 다른 기기/웹에서의 조회
- 서비스 품질 개선, 결제·구독 관리, 무료 플랜 광고 제공

## 3. 보관 및 파기
- 로컬 데이터는 사용자가 앱에서 삭제하거나 앱을 제거하면 함께 삭제된다.
- 클라우드 업로드 데이터(동의 시)는 사용자가 삭제 요청하거나 계정 해지 시 파기한다.

## 4. 제3자 제공·처리위탁
- 본인 외 제3자에게 데이터를 판매·제공하지 않는다.
- 처리위탁: Google Firebase(클라우드 저장, 동의 시), Google Play Billing(결제), Google AdMob(광고).
- 로컬 네트워크(WiFi) 스트리밍: 동일 네트워크의 사용자 기기로만 직접 전송되며 외부 서버를 거치지 않는다.

## 5. 권한 고지
- **마이크(RECORD_AUDIO):** 소음 측정·녹음. 거부 시 핵심 기능 사용 불가.
- **위치(ACCESS_FINE/COARSE_LOCATION):** 측정 장소 기록. 거부해도 측정은 가능(위치만 미기록).
- **알림(POST_NOTIFICATIONS):** 상시 측정 상태 표시.
- **포그라운드 서비스(마이크):** 장시간 측정 유지.

## 6. 이용자 권리
- 데이터 열람·삭제·수집 동의 철회 가능. 앱 내 삭제 또는 문의처를 통해 요청.

## 7. 아동
